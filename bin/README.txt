Beim Programmstart gibt es oben ein graues und unten ein weißes Fenster.
Man kann unten lateinischen Text tippen oder hineinkopieren.
Dieser Text wird in Runen übersetzt. 
Man kann über Kopieren den geschriebenen Text in die Zwischenablage kopieren.
Man kann über Speichern den geschriebenen Text abspeichern. 
Man kann Runen / Lateinisch(e Buchstaben) lesen, beim lesen werden diese automatisch in die andere Schrif tübersetzt.
R. einfügen: Man kann Runen in ein separates Fenster einfügen und diese in lateinische Lettern übersetzen lassen.
